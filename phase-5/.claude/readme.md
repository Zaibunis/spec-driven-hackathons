# Claude Skills

A repository containing various skills and tools for AI agents, focused on enhancing capabilities in areas like document processing, automation, and development.

## Project Structure

The project is organized into various skills located in `claude-code-skills-lab/.claude/skills/`.

### Available Skills

- **browser-use**: Automate browser interactions, form filling, and data extraction using Playwright.
- **context7-efficient**: Token-efficient library documentation fetcher using Context7 with intelligent filtering.
- **doc-coauthoring**: Structured workflow for collaborative document creation (specs, proposals, decision docs).
- **docx**: Comprehensive tools for creating, editing, and analyzing Word documents (.docx), including redlining.
- **internal-comms**: Templates and guidelines for writing effective internal communications (newsletters, updates).
- **nextjs**: Best practices, patterns, and checklists for Next.js App Router development.
- **openai-agents-python**: Token-efficient documentation fetcher for OpenAI Agents Python SDK (multi-agent workflows, handoffs, guardrails, tracing).
- **openai-chatkit**: Framework for building AI-powered chat experiences with ChatKit (streaming, theming, tools, React/Python).
- **pdf**: Toolkit for manipulating PDFs, including text extraction, creation, form filling, and merging.
- **pptx**: Tools for creating and editing PowerPoint presentations, including HTML-to-PPTX conversion.
- **python-code-writer**: Guidelines and conventions for writing clean, idiomatic Python code.
- **skill-creator**: A guide and set of tools for creating, initializing, and packaging new Claude skills.
- **theme-factory**: A collection of professional themes (colors/fonts) for styling artifacts like slides and docs.
- **xlsx**: Scripts and guidelines for creating, analyzing, and manipulating Excel spreadsheets with formula support.
